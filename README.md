## 🚀 About Unfee

Unfee is a Telegram-based trading bot built for the Solana ecosystem.  
It allows users to trade tokens directly from Telegram with **zero trading fees**, providing a fast, secure, and seamless experience.

Designed for efficiency and simplicity, Unfee removes unnecessary costs while maintaining powerful trading capabilities for the Solana community.

## ✨ Key Features

- ⚡ **Zero Trading Fees** – Trade with absolutely no platform tax.
- 🤖 **Telegram Native Bot** – Trade directly from Telegram.
- 🔒 **Secure Transactions** – Built with safety and reliability in mind.
- 🌐 **Solana Ecosystem** – Optimized for Solana’s high-speed blockchain.
- 💡 **Simple UX** – Easy interface for both beginners and advanced traders.

## 🧩 How It Works

1. Open the Unfee Telegram bot.
2. Connect your Solana wallet.
3. Select the token you want to trade.
4. Execute buy or sell orders instantly.
5. Enjoy **zero platform fees** on every trade.
